# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-16

### Added
- Timer Pomodoro circular com modos de foco e pausa
- Seletor de modo (foco, pausa curta, pausa longa)
- Controles de timer (iniciar, pausar, reiniciar)
- Gerenciamento de atividades vinculadas às sessões
- Página de relatórios com gráficos de produtividade via Recharts
- Ícone na bandeja do sistema (tray) com menu de contexto
- Banco de dados local SQLite via sql.js (WebAssembly)
- Suporte a temas com Material Design 3 (MUI v5)
- Navegação por Navigation Rail
- Persistência de configurações via electron-store
